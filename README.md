# cYHBernews 🚀

Periódico de tecnología y ciberseguridad con estética premium inspirada en **iOS / macOS**.

## 🌐 Enlace del Proyecto
Puedes ver la página en vivo aquí:  
👉 **[https://devcop95.github.io/cYHBernews/](https://devcop95.github.io/cYHBernews/)**

## ✨ Características
- **Interfaz iOS:** Diseño ultra-compacto y minimalista con estética de Apple.
- **MacBook Console:** Noticias presentadas como código Python dentro de una ventana de consola.
- **Fondo Dinámico:** Ondas azules monocromáticas optimizadas para alto rendimiento.
- **Carga Dinámica:** El sitio lee automáticamente el archivo `noticias.json`.
- **Filtros Inteligentes:** Navegación por categorías mediante pestañas estilo iOS.
- **100% Estático:** Rápido, ligero y fácil de hospedar en GitHub Pages.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Keyframes & Glassmorphism)
- JavaScript (Vanilla)
- Lucide Icons
- Google Fonts (Inter & JetBrains Mono)
