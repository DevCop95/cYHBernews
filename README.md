# cYHBernews 🚀

Periódico de tecnología y ciberseguridad con estética premium inspirada en **iOS / macOS**.

## 🌐 Enlace del Proyecto
Puedes ver la página en vivo aquí:  
👉 **[https://devcop95.github.io/cYHBernews/](https://devcop95.github.io/cYHBernews/)**

## ✨ Características Principales
- **Interfaz iOS Premium:** Diseño ultra-compacto, minimalista y elegante con efectos de *glassmorphism* y desenfoque (blur) nativo.
- **Menú de Herramientas Inteligente:** Panel desplegable que incluye búsqueda en tiempo real, ordenación avanzada y estadísticas del feed.
- **Gestión de Menú Mejorada:** 
    - Cierre automático tras realizar búsquedas o filtrar.
    - Indicador de estado visual mediante intercambio de iconos (Menú / X).
    - Soporte para cierre mediante clic exterior y tecla Escape.
- **Fondo Dinámico 'Mesh Gradient':** Gradientes animados de alto rendimiento que crean una atmósfera tecnológica inmersiva.
- **Carga Dinámica:** Motor de renderizado en JavaScript que consume `noticias.json` con soporte para caché mediante versionado.
- **Secciones de Actualidad:** Organización automática de noticias por días y categorías (Actualidad, Seguridad, Tech, IA).
- **Diseño Responsivo Total:** Optimizado específicamente para dispositivos móviles con navegación táctil fluida.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5 semántico y CSS3 avanzado (Variables, Flexbox, Grid, Animations).
- **Lógica:** JavaScript ES6+ (Manipulación de DOM, Fetch API, Normalización de texto).
- **Iconografía:** Lucide Icons para una estética vectorial limpia.
- **Tipografía:** Inter (UI) y JetBrains Mono (Código/Titulares) vía Google Fonts.

## 📁 Estructura del Proyecto
- `index.html`: Estructura principal y componentes UI.
- `app.js`: Lógica de filtrado, búsqueda, gestión de estado y animaciones.
- `style.css`: Estilos, animaciones de fondo y adaptabilidad móvil.
- `noticias.json`: Base de datos estática de artículos.
